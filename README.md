## Other
- Show Cursor
    - `?cursor=1`
- Random
    - `?rand=1`
- Random With Custom Color Scheme
    - `?randc=X` (replace `X` with the flag's id you want it to use.)
- Custom Hex Color
    - `?hex=ffffff` (use whatever hex code you want.)
